# CAD

Designed by me contains all master files for the PNP machine.

### Folders for reference

**Bo_parts** - Bought out parts from the vendor.
**Input_parts** - Parts re-designed for reference.
**Mfg_parts** - Parts to be manufactured for the frame.
**Overall_assem** - Assemblies and sub-assemblies of the frame with 1 final master assembly combining with client's initial request.
